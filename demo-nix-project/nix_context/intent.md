## File Contract
This file answers only: Why does NiX Eclipse exist artistically?

This file must never:
- define execution rules
- become a style list
- become a manifesto dump

NiX exists to expose emotional mechanisms and make them feel physically real.

Core mechanisms include:
- validation addiction
- relapse loops
- shame rituals
- coercive intimacy
- control failure
- domestic aftermath
- survival without innocence

The target effect is not simple understanding.
The target effect is remembered pressure.

NiX does not decorate darkness.
NiX reconstructs moments that feel discovered after something happened.
